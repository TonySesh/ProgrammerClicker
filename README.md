Моя первая игра на Unity - симулятор программиста (кликер)

Идея: кликаешь - получаешь опыт, за опыт покупаешь навыки, навыки дают дополнительные клики, благодаря дополнительным кликам заработок идёт быстрее, за деньги покупаешь интерьер.

Почти все текстуры рисовались собственноручно.
